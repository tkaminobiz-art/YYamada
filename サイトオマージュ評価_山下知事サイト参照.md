## 1. 参考サイトの分析結果（Site Homage ステップ1）

| 項目 | 山下知事サイトの仕様 |

### 1-6. 雰囲気（まとめ）

- **第一印象:** 緑を基調にした、信頼感のある政治サイト。本人の顔と数字がはっきり見える。
- **一貫性:** 本人写真・名前・数字が繰り返し出てきて「ひとりのリーダー」が伝わる。
- **余白:** 多め。詰め込み感がない。

## 2. 現在の山田洋平サイトの状況

- **技術:** HTML/CSS/JS 静的サイト。Noto Sans JP / Noto Serif JP。CSS変数で緑・ティール・バー色を定義。
- **構成:** ヒーロー（本人写真＋グラデ＋名前）→ 新着情報バー＋200万テイザー＋キャッチ＋バッジ3つ＋LINE CTA → ごあいさつ → 動画・SNS（SNSアイコン＋YouTube1本）→ 政策（バー＋01〜05）→ 実績（2大ボックス 200万/125万＋カードリスト）→ 議会活動 → 活動 → プロフィール → 後援会・LINE → フッター（ダークティール）。
- **画像:** ヒーロー・プロフィールは山田氏本人（1.png, 2.png）。ごあいさつ・三つの約束・政策・議会・活動には sty_family, msg_children, sty_scenery, asm_question, act_* を使用（一部は実装計画で不適切と指摘済み）。

## 4. 総合評価と次のアクション

### オマージュ解像度（5段階イメージ）

- **レイアウト・構成:** ★★★☆☆ … セクションの種類は揃っているが、順序・ブロックの型（93%バー、政策3ボックス＋矢印、ニュース、動画2本、フッターSNS）が参照と完全には一致していない。
- **配色・タイポ:** ★★★★☆ … 緑・白・ダークのトーンは近似。フォント・数字の大きさはやや差あり。
- **コンポーネント・余白:** ★★★☆☆ … 2大ボックス・バーは参照に近い。バッジ3つ＋LINE単体は紋切り型で参照にはない。
- **画像・コンテンツ適切性:** ★★☆☆☆ … 本人以外の人物写真・未許諾の可能性がある画像が残っており、政治サイトルールと実装計画の指摘に未対応。

### 推奨アクション（優先順）

1. **画像の整理** … 実装計画 Phase 0 に従い、本人・許諾以外の人物写真を外す。政策・議会・ごあいさつ・活動の不適切画像を差し替え or 画像なしに。
2. **構成の参照への寄せ** … 「公約達成」相当で93%風のバー or 数字1本のインパクト、政策ブロックで「3ボックス＋矢印＋01〜07」の流れ、県政ニュース相当のブロック、動画2本 or SNSラベル付き、フッターにSNS＋TOP を検討。
3. **紋切り型の削減** … バッジ3つ横並び＋LINE単体CTA をやめ、バッジの数・形の差別化、CTA の多様化（後援会・お問い合わせ・QR等）を実施。
4. **デザイン言語化仕様の継続利用** … `山下知事サイト_デザイン言語化仕様.md` と付録A（実HTML構造）を実装の正本として、次回のフルリニューアル時に反映する。
