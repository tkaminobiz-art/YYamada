## 第1部：コンテキスト定義（Phase 0）

### 1-1. ターゲット

| 項目 | 内容 |
|------|------|
| **主ターゲット** | 奈良県生駒市および県内の有権者（30〜60代、子育て世代・経営者・公務員・無党派層） |
| **心理状態** | 「誰に投票すれば奈良が良くなるか」の比較検討中。数字と実績で納得したい層と、地元愛・人柄で共感したい層の両方 |
| **意思決定** | 選挙時の投票行動＋日常的な「この人なら託せる」という信頼の蓄積 |

### 1-2. サイトの目的

| 目的 | 優先度 | 指標 |
|------|--------|------|
| **CV（コンバージョン）** | 最優先 | LINE登録・後援会入会・問い合わせ |
| **認知・ブランディング** | 高 | 「公認会計士の眼で奈良の家計を立て直す」という一貫したメッセージの浸透 |
| **信頼の可視化** | 高 | 身を切る改革・議会追及・生駒の現場活動の「証拠」を残す |

### 1-3. 業種・価格帯

- **業種:** 政治活動・議員公式サイト（BtoC＝有権者向け）
- **トーン:** 誠実・論理的・情熱的。煽りや過度な誇張は禁止。証拠（数字・実績）で説得。

### 1-4. ベンチマーク

- **参考サイト:** 星川大地 公式サイト（https://www.daichi-star.com/）
  - 同様のターゲット（奈良県民・維新系・改革訴求）、同一政党、セクション構成の参考として有効。
- **絶対評価の物差し:** Awwwards SOTD 8.0／「テンプレート品質」でないこと。

## 第2部：参考サイト（星川大地）診断

参考サイトを Design Critic に基づき簡易診断し、山田サイトで「踏襲する点」と「改善する点」を明確にする。

### 2-1. 3秒テスト（第一印象）

| 項目 | 評価 |
|------|------|
| **3秒スコア** | 7/10 |
| **第一印象** | 政治家サイトとしては整理されているが、やや「テンプレート的」なセクション積み上げ。 |
| **離脱衝動** | FVのコピー「事件が起きてから動くのではなく…」は強い。画像・CTAの配置で離脱を防ぐ余地あり。 |

### 2-2. 5軸採点（簡易）

| 軸 | スコア/20 | 所見 |
|----|-----------|------|
| 美しさ | 14 | 余白・タイポのメリハリはある。色数・背景の統一をさらに強めるとプレミアムに。 |
| 楽しさ | 12 | スクロール連動・スタガー等の「目的あるアニメ」を増やすと差別化できる。 |
| おしゃれさ | 13 | 2026トレンド（Bento・ノイズ・キネティックタイポ）の適用余地あり。 |
| 惹きつける力 | 15 | VISION・RESULTS・ASSEMBLY の構造はCVRに有効。CTAの補色・マイクロコピーを強化。 |
| 技術的実装 | 14 | レスポンシブ・情報構造は良好。LCP・フォント最適化の確認を推奨。 |
| **合計** | **68/100** | **Bランク（プロの仕事。良いが記憶に残るまでには一歩不足）** |

### 2-3. 山田サイトで「踏襲する」要素

- **セクション順:** Hero → キャッチ／ビジョン → 政策（重点） → 実績（数字） → 議会活動（Q&A形式） → ロードマップ／活動報告 → ストーリー → プロフィール
- **LINE CTA の prominence:** ファーストビュー付近および固定フッターでの露出
- **「結論ファースト」:** 冒頭でベネフィット（なぜこの人なのか）を明示

### 2-4. 山田サイトで「改善する」要素

- **FVの3秒ルール徹底:** ベネフィットコピー＋権威バッジ（公認会計士・身を切る改革の数字）＋メイン画像＋CTA を必ず揃える
- **色・タイポのトークン化:** 7色以内・80/15/5・オフホワイト背景・和文 weight 500 以下
- **同一レイアウトの反復を避ける:** Bento／非対称／スティッキー等でパターンに変化をつける
- **CTA:** 補色使用・44px以上・「LINE登録」「後援会入会」等の具体的ラベル＋リスクリバーサル（無料・簡単）

## 第3部：山田洋平サイト 設計方針

### 3-1. LP-OS に基づくフレームワーク選択

- **ターゲット層:** 潜在層（「維新っぽい人」程度の認識）と顕在層（比較検討中）の両方
- **採用フロー:** **結論ファースト** ＋ **PASONA 的流れ**（課題の言語化 → 親近感 → 解決策＝山田の政策・実績 → オファー＝LINE・後援会 → 行動）
- **7-Step 対応:**
  - **Interest（結）** … FV「公認会計士の眼で、奈良の家計を立て直す。」＋ 身を切る改革の数字
  - **Problem（起）** … 「税金の無駄」「次世代への負の遺産」「生駒の交通・子育て」
  - **Evidence（承）** … 政務活動費200万円返還、報酬125万円以上削減、NAFIC・県立病院機構追及
  - **Empathy（転）** … 生駒の子→生駒の父、2児の父・猫、俵口小・生駒中のストーリー
  - **Authority（転）** … 公認会計士・税理士、あずさ監査法人、維新八策
  - **Passion（転）** … 「人生逃げ場なし」、次世代への責任
  - **Action（結）** … LINE登録・後援会・問い合わせ CTA

### 3-2. USP（核メッセージ）

- **キャッチコピー案（報告書準拠）:**
  - メイン: **「公認会計士の眼で、奈良の家計を立て直す。」**
  - サブ: 「人生逃げ場なし。次世代に責任を持つ、覚悟の政治。」
  - 地域: 「生駒で育ち、生駒で親となり、生駒の未来を創る。」

- **差別化の核:** 「資本市場の番人」としての分析力 × 二児の父・生駒育ちの温かさ。数字で証明する改革と、地元を歩く人間味の両立。

### 3-3. 感情導線（ストーリーテリング）

1. **共感** … 生駒の子→父、子供の未来への問い
2. **驚き・納得** … 200万円返還・報酬削減・NAFIC追及等の「数字」
3. **信頼** … 会計士資格・議会質問・委員会活動
4. **行動** … LINE・後援会（無料・簡単の明示）

## 第5部：デザイン・ビジュアル方針

### 5-1. 視覚的アイデンティティ（報告書＋Design System）

| 項目 | 方針 |
|------|------|
| **メインカラー** | 維新のグリーンを基調。信頼・誠実のネイビー、白（オフホワイト）を効果的に配置。 |
| **使用比率** | ニュートラル 80% / メイン（グリーン・ネイビー）15% / アクセント（CTA）5% |
| **背景** | オフホワイト（#FAFAF7 等）。純白 #FFFFFF 禁止。 |
| **テキスト** | ソフトブラック（#1C1C1C〜#222）。真黒 #000 禁止。 |
| **タイポグラフィ** | 知的＝明朝体、力強さ＝ゴシック体の使い分け。和文 font-weight は最大500まで。見出し／本文サイズ比 3倍以上。 |

### 5-2. デザイントークン（案）

```
--primary:     #2E5E3A  (維新グリーン・メイン)
--accent:      #E85D04  (CTA用・オレンジ系) または 補色で目立つ1色
--bg:          #FAFAF7  (背景)
--text:        #1C1C1C  (本文)
--text-light:  #777777   (補助)
--navy:        #1B365D   (信頼・サブ)
```

- **フォント:** 見出し＝Noto Serif JP または 明朝系 / 本文＝Noto Sans JP / 数字＝Inter (tabular-nums)
- **スペーシング:** 8px グリッド。セクション間 120〜200px。コンテンツ幅 max 1200px。
- **ボタン:** タップ領域 高さ44px以上。border-radius 4px（999px禁止）。transition 400ms cubic-bezier(0.16,1,0.3,1)。

### 5-3. 写真・ビジュアル

- **3つの顔のバランス:** ①議会で鋭い眼差しの「プロの顔」 ②生駒の街頭・市民と笑顔の「地元の顔」 ③子供・猫と過ごす「柔らかな顔」
- **禁止:** グラデーションのプレースホルダー公開、AI生成の完璧すぎる画像のそのまま使用。実写・現場感を優先。

### 5-4. 2026トレンドの適用

- **Bento UI:** 実績（RESULTS）や活動報告を不均等グリッドで1セクション以上に適用。
- **マイクロインタラクション:** CTA ホバーでリフト・グロー。カードは translateY(-4px)。
- **アニメーション:** 400〜800ms、ease cubic-bezier(0.16, 1, 0.3, 1)。スタガー 50ms。prefers-reduced-motion 対応。

## 第6部：CTA・EFO 最適化

| 要素 | ルール |
|------|--------|
| **次元** | モバイルでタップ領域 高さ44px以上を死守。 |
| **マイクロコピー** | 「無料」「簡単」「本人とつながる」等で心理的ハードルを下げる。 |
| **色** | メインの補色でCTAを目立たせる。 |
| **ラベル** | 「公式LINE登録」「後援会入会はこちら」等、具体的で低負荷なアクション。 |
| **配置** | FV に必ず1つ。セクション終わり・PROFILE 前後に反復。固定フッターまたはスティッキーバーを推奨。 |

## 第8部：検証チェックリスト（公開前）

### LP-OS

- [ ] 結論ファーストで FV にベネフィット・権威・画像・CTA が揃っているか
- [ ] 感情導線（共感→驚き→信頼→行動）が設計されているか

- [ ] 3秒で「この人なら託せる」の手がかりが伝わるか
