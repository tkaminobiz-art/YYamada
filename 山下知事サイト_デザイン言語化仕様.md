# 山下知事サイト（yamashitamakoto.com）デザイン言語化仕様

**目的:** スクリーンショットを隅まで見てデザインを言語化し、山田洋平サイトのオマージュ解像度を最高レベルにする。  
**使い方:** 実装時はこの仕様を正本とし、ピクセル・トーン・構成を可能な範囲で再現する。  
**出典:** 山下知事サイト実機のソース（View Source）を反映した構造メモを「付録A」に記載。

---

## 0. 全体のトーン・原則

- **色の主役:** 緑（複数トーン）。白・ライトグリーン・ダークグリーンが層になっている。
- **人物:** 本人（山下知事）の肖像が、ヒーロー・カード・動画サムネ・政策ブロックで**一貫して**使われている。他人・ストックは使っていない。
- **情報の優先度:** 数字（373億円・4,620件・93%など）が大きく、セクションの軸になっている。
- **余白:** セクション間はたっぷり。ブロック内は読みやすい行間・パディング。

## 3. 新着情報／最新情報セクション

| 項目 | 仕様 |
|------|------|
| **セクション見出し** | 「最新情報」または「新着情報」。 **見出しの右側:** 小さな緑のボタン「一覧」。 |

## 6. 実績・数字の2大ボックス

| 項目 | 仕様 |
|------|------|
| **セクション見出し** | 例「山下知事のこれまでの実績をご紹介します！」。 |
| **背景** | 薄いミントグリーン／ライトグリーン。 |
| **ボックス** | **2つ横並び**。同じ幅。 **各ボックス:** **ヘッダー:** 上にダークグリーンの細い帯。帯の中にラベル（例「奈良県政の課題」／「改革の成果」）。文字は白。 **数字:** その下に**極めて大きい**数字。「373億円」「4,620件」。フォントはゴシック・太字。色は緑 or 黒。 **補足:** 数字の下に小さな説明文。 **アイコン:** 小さな緑の下向き矢印など、あれば。 |
| **角丸** | ボックス全体は 8〜12px。ヘッダー帯は上だけ角丸。 |

## 12. オマージュ解像度を上げるためのチェックリスト

実装前に、以下で仕様との対応を確認する。

- [ ] ヘッダーは白背景・左ロゴ緑・右ナビか。
- [ ] ヒーローは「左: 緑＋名前＋サブコピー / 右: 本人写真」の構成か（SPは縦積み可）。
- [ ] 新着情報は薄い緑の帯＋見出し＋カード or テキストか。
- [ ] 動画は「SNS＋動画」の順で、サムネ or 埋め込みか。
- [ ] 実績は「2大ボックス（ダークヘッダー＋大数字）」か。
- [ ] 政策は 01〜07（or 01〜05）のグリッドで、番号・タイトル・詳しく見るか。
- [ ] ごあいさつは**1回だけ**で、本人 or 許諾画像 or なしか。
- [ ] プロフィールは本人写真（円形 or 矩形）＋経歴か。
- [ ] フッターは最下段がダークグリーン全幅で名前・著作権か。
- [ ] 色は緑のトーン＋白＋ダークグリーンが主で、アクセントは控えめか。
- [ ] 本人以外の人物写真は使っていないか。

**次のステップ:** この言語化仕様を実装計画の「Phase 0 の前」に置き、実装時は本仕様を参照してオマージュの解像度を最大にする。画像の整理（Phase 0）は、このデザイン仕様とセットで進める。
